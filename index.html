Summary: Latitude vs. X

The purpose of this project was to compare the weather at different latitudes in order to understand how they change as we get closer to or farther away from the equator. We then draw conclusions about the behavior of the climate and whether or not it seems to have a correlation with latitude. We compare facets such as wind speed, temperature, cloudiness, and humidity. We used OpenWeatherMap's API in order to collect weather information from approximately 500 random cities around the world, and assembled them into a dataset.

We used matplotlib to plot the different aspects of the weather vs. latitude. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations that we witnessed.
